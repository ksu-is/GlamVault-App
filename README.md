# GlamVault-App

GlamVault is a beauty product inventory and expiration tracker designed for people who love makeup, skincare, haircare, fragrance, and beauty products.

## Purpose

Many people forget what products they already own, accidentally buy duplicates, or use expired products. GlamVault helps users stay organized by tracking beauty products, expiration dates, spending, and wishlists.

## Features

- Product inventory list
- Expiration reminders
- Budget tracker
- Wishlist section
- Search and filter options
- Duplicate product alerts
- Photo uploads
- Favorite products list

## Target Users

- Beauty influencers
- Makeup lovers
- College students
- Skincare enthusiasts
- People with large beauty collections

## Tools and Technologies

- Python
- Tkinter or Flask
- SQLite or CSV database
- GitHub Desktop
